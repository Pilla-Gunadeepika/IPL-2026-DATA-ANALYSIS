# IPL 2026 Data Analysis Using Python

This project analyzes IPL 2026 match data using Python, Pandas, NumPy, and Matplotlib. The notebook explores team performances, batting statistics, bowling statistics, and venue-based insights through data visualization.

## Analyses Performed

Match Winners Analysis

Top 10 Run Scorers

Players with Most Sixes

Players with Most Fours

Top Wicket Takers

Team-wise Total Runs

Venue-wise Total Runs

Venue Hosting Most Matches

Team-wise Sixes

Team-wise Wickets

## Technologies Used

Python

NumPy

Pandas

Matplotlib

Google Colab

## Note

The dataset used in this project contains only the initial matches of the IPL 2026 season and does not represent the complete tournament.
